Chapter 4: Networks
Term Definition
Network Classifications:
  • LAN:
  • MAN:
  • WAN:
  • Group of interconnected computers in a single buildinng or facillity
  • A medium size network; typically spans city/town wide
  • A worldwide network; spans across countries.
Network models:
  • Bus network:
  • Star network:
  • All machines are connected to a shared communication station called a bus.
  • All the machines in the network are connected to a central machine, and communicate through it..
Protocols:
  • CSMA/CA Ethernet:
  • CSMA/CA Wifi:
  • Internet protocol that allows machines to send signals via a bus network
  • Internet protocol that prevents collisions between machines; collisions lead to the retransmission of the signal
Combining networks:
  • Repeater:
  • Bridge:
  • Switch:
  • Router:
  • Gateway:
  • A device that allows communication between two bus networks.
  • Similar function to a repeater; but only sends signal across network when destination is determined
  • A bridge that connects multiple networks
  • A specialized machine that forwards signals from source to other networks.
  • Connection point between a network and an internet.
Client/server model:
  • P2P:
  • Distributed system:
  • Cloud-computing:
  • Peer 2 peer model; Information is shared directly from one user on an internet to another.
  • A distributed system of computing that allows computers on a network to distribute a complex task into smaller parts.
  • Large set of super computer(s) that has the sole function allocated for people on a network/internet
Internet adressing:
  • Tier 1:
  • Tier 2:
  • Tier 3:
  • International WANS that operate at a highspeed; servers as the backbone of the Internet
  • Regional Wans operating at a slower speed then tier 1
  • Provided by ISPs allows user to connect to tiers 1 and 2
IP address:
  • ICANN:
  • DNS:
  • TLD:
  • Country-code TLD:
  • Organization that allocates ip adresses
  • Directory for domain names on the internet
  • Reflects the purpose of a domain ex:.edu, .gov
  • TLDs that are specific to different countries
Internet Protocols:
  • FTP:
  • HTTP:
  • SMTP:
  • An Application use to accsess another machines files via a network/internet
  • Modern Internet protocol used for web browsing
  • Protocol that allows emails between two machines.
World Wide Web:
  • URL:
  • Client-side activities:
  • Server-side activities:
  • Adress given to sites, and different elements of sites that allows user with adress to accsess data as needed
Internet four layer software:
  • Application layer:
  • Transport layer:
  • Network layer:
  • Link layer:
  • Applications that make use of servers for interent based communication
  • Sends and recieves messages via internet
  • Determines the destination of packets sent in each step
  • Transfers the packets to a network and works around the type of network its sent to
Internet is:
  • Packet-switching network:
  • Fault tolerant:
  • Redundant:
  • Data is split into smaller pieces called packets
  • Ability to keep network communication running even when parts fail
  • Increases reliabilty because there are alternatives to any part that may fail.
Cybersecurity:
  • Malware:
  • Virus:
  • Worm:
  • Trojan horse:
  • Spyware:
  • Phishing:
  • DDOS attack:
  • Stands for malicious software; includes any software designed for malicious intent.
  • A self replicating computer program
  • A trojan horse capable of replicating through a network once it has infected one machine on the network
  • A type of virus that is snuck into another program; it is allowed in by a human operator
  • A type of malware used to track peoples mouse movements, key strokes, and other input data.
  • Emails or other means of communication designed to look like its from an official source; used to steal login info
  • Distributed Denial of Service attack; Uses packets sent from multiple machines on a botnet to disrupt online services
Protection:
  • Firewall:
  • Spam filter:
  • Proxy server:
  • Antivirus:
  • Cryptography:
  • Public-key encryption:
  • Regulates incoming and outgoing traffic in a network; based on predetermined protocols.
  • Filters messages via emails, and sorts them. Goal is to prevent spam aka junk messages typically promoting a scam or shady business
  • Acts as in intermediate server between clients and providers.
  • Prevents, detects, and removes malware
  • Creating secure comms via encryption. Only a person with an encryption key can read the message.
  • Anyone can encrypu using the public key, but the messages can only be decrypted with a private key; held by one person.
